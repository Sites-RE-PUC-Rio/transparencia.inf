Título:

 Empréstimos de exemplares de livro.

Objetivo:

 Controlar o empréstimo de exemplares de livros.
Contexto: Biblioteca emprestando exemplares aos usuários comuns.
Atores: sistema, usuário comum e usuário empregado.
Recursos: livro, exemplar, cartão de usuário
Episódios:

1. Usuário comum solicita empréstimo de livro a usuário empregado, mostrando seu cartão de identificação.

2. Usuário empregado solicita ao sistema exemplar disponível do livro pedido.

2. Sistema verifica se existe exemplar disponível do livro pedido.

3. Se não existe exemplar disponível do livro pedido, usuário comum é informado e não recebe exemplar do livro.

4. Se existe exemplar disponível do livro pedido:

4.1. Usuário empregado informa ao sistema número de identificação do usuário comum.

4.2. Sistema verifica número de exemplares emprestados do usuário comum.

4.1.1 Se número de exemplares emprestados de usuário comum é maior que o limite de exemplares emprestados: informa ao usuário comum que o limite de exemplares emprestados foi excedido.

4.1.2. Se número de exemplares emprestados de usuário comum é menor que o limite de exemplares emprestados:

4.1.2.1. Sistema desmarca exemplar como exemplar disponível.

4.1.2.2. Sistema marca exemplar como exemplar emprestado a usuário comum.

4.1.2.3. Sistema incrementa número de exemplares emprestados a usuário comum.

4.1.2.4. Sistema informa a usuário empregado que exemplar foi emprestado.

4.1.2.5. Usuário empregado entrega o exemplar ao usuário comum.

Título:

 Devolução de exemplares de livro.

Objetivo:

 Administrar os exemplares de livros devolvidos.
Contexto: Usuários comuns devolvendo exemplares de livros à biblioteca.
Atores: sistema, usuário comum e usuário empregado.
Recursos: livro, exemplar
Episódios:

1. Usuário comum devolve um exemplar emprestado a usuário empregado.

2. Usuário empregado solicita devolução de exemplar ao sistema.

3. Sistama verifica prazo de devolução do exemplar.

            3.1. Se prazo de devolução do exemplar expirou:

                        3.1.1. Sistema calcula multa e informa a usuário empregado.

                        3.1.2. Usuário empregado solicita a usuário comum o pagamento da multa.

                        3.1.3. Usuário comum paga a multa.

                        3.1.4. Usuário empregado informa ao sistema que a multa foi paga.

4. Sistema marca exemplar como exemplar disponível.

5. Sistema desmarca exemplar como exemplar emprestado a usuário comum.

6. Sistema decrementa número de exemplares emprestados a usuário comum.

7. Sistema informa a usuário empregado que exemplar foi devolvido.

8. Sistema emite a comprovante de devolução.

9. Usuário empregado entrega comprovante de devolução a usuário comum.

10. Usuário empregado recoloca exemplar em seu lugar.

Título:

 Remoção de exemplar.

Objetivo:

 Controlar a remoção de exemplares no acervo.
Contexto: Biblioteca está diminuindo o acervo.
Atores: usuário empregado e sistema
Recursos: exemplar
Episódios:

1. Usuário empregado fornece ao sistema número de registro do exemplar a ser removido.

1. Sistema verifica se exemplar a ser removido existe.

1.1. Se exemplar a ser removido não existe, sistema informa a usuário empregado que exemplar a ser removido não existe.

1.2. Se exemplar a ser removido existe:

1.2.1. Sistema exclui o registro correspondente ao exemplar em questão.

1.2.2. Sistema informa a usuário empregado que exemplar foi removido do acervo da biblioteca.

Título:

 Adição de exemplar em acervo.

Objetivo:

 Controlar a adição de exemplares no acervo.
Contexto: Biblioteca está expandindo o acervo.
Atores: usuário empregado e sistema
Recursos: livro, exemplar
Episódios:

1. Usuário empregado fornece ao sistema o ISBN do livro correspondente ao exemplar a ser adicionado.

2. Sistema verifica se o livro com o ISBN fornecido já se encontra registrado.

2.1. Se ISBN fornecido não se encontra registrado, sistema solicita ao usuário empregado que adicione o livro em questão, sendo que o sistema já fornece automaticamente ao processo de adição de livros o ISBN do livro em questão.

3. Sistema informa a usuário empregado que exemplar foi adicionado ao acervo da biblioteca.

4. Sistema imprime ficha com número de registro do exemplar

5. Usuário empregado cola ficha na contra-capa do exemplar.

6. Usuário empregado coloca o novo exemplar no lugar a ele reservado.

Título:

 Remoção de livro.

Objetivo:

 Realizar a remoção de livros no acervo.
Contexto: Biblioteca está retirando um livro.
Atores: usuário empregado e sistema
Recursos: livro
Episódios:

1. Usuário empregado fornece ao sistema o ISBN do livro a ser removido.

2. Sistema procura livro com ISBN fornecido.

2.1. Se livro com ISBN fornecido não se encontra registrado, usuário empregado é avisado.

2.2. Se livro com ISBN fornecido já se encontra registrado, sistema verifica quantos exemplares do livro em questão estão registrados.

2.2.1. Se ISBN fornecido já se encontra registrado e não há exemplares do livro em questão:

2.2.1.1 Sistema remove registro do livro.

2.2.1.2. Sistema informa a usuário empregado que livro foi removido do acervo da biblioteca.

2.2.2. Se ISBN fornecido já se encontra registrado e há exemplares do livro em questão, sistema informa a usuário empregado que não é possível a remoção do livro enquanto houverem exemplares deste livro registrados.

Título:

 Adição de livro.

Objetivo:

 Controlar a adição de livros no acervo.
Contexto: Biblioteca está expandindo a quantidade de livros no acervo.
Atores: usuário empregado e sistema
Recursos: livro, Restrição: deve ter ISBN
Episódios:

1. Usuário empregado fornece ao sistema o ISBN do livro a ser adicionado.

2. Sistema verifica se o livro com o ISBN fornecido já se encontra registrado.

2.1. Se ISBN fornecido já se encontra registrado, usuário empregado é avisado que livro não foi adicionado.

2.2. Se ISBN fornecido não se encontra registrado:

2.2.1 sistema solicita ao usuário empregado o título, autor, área de interesse, editora e edição do livro em questão.

2.2.2. Sistema adiciona registro do livro.

2.2.3. Sistema informa a usuário empregado que livro foi adicionado ao acervo da biblioteca.
